# stream-frontend — Marketing Site Assets

Static assets for the GitHub Pages landing site at
`https://akshatsinghkaushik.github.io/stream-frontend/`.

## Files

| File | Purpose |
|---|---|
| `favicon.svg` | Browser tab icon and PWA icon |
| `og-image.svg` | Open Graph / Twitter Card share image (1200 x 630) |
| `robots.txt` | Search engine crawl permissions |
| `sitemap.xml` | Page index for search engines |
| `site.webmanifest` | PWA manifest (theme colour, display mode, icons) |

## Design System

- Background: `#0B1120` / Surface: `#0F172A`
- Primary: `#E11D48` (rose) / Secondary: `#FB7185`
- Accent / CTA: `#2563EB` (blue)
- Foreground: `#F8FAFC` / Muted: `#1E293B` / Border: `#334155`
- Display font: **Sora** / Body font: **Inter**

The full landing page is `docs/index.html` (single-file, Tailwind Play CDN, no build step).
